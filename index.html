Web Work

This website hosts a collection of web projects I've worked on; the good, the bad, and the ugly.

Ongoing

These are projects I still tinker with, they may be spun off into separate repos later.

Where I've Been

Map which uses SVG-CSS integration to show labels on hover. It pulls precisely zero clever tricks to achieve this effect, all done with explicit labelling. Would like to make it smarter in future.

OS Timeline

An in-progress idea to create timelines from JSON files to be rendered as HTML. There are some existing packages which can do this but most are premium and/or not open source.

Society Timeline

Another website timeline project, this one being used for my undergrad uni society membership. The stylesheet for this one was much nicer but I never got around to making a generator.

Archive

Democracy Project

Back in summer 2014 I had an idea to have an icon theme whereby all changes were voted on. There was some initial interest but it quickly fizzled out.

Timezone Tracker

Back when Numix was a big team which worked (on occasion) synchronously, we used this page to keep track of what time it was where everyone was. It simply uses timeanddate.com iframes.
